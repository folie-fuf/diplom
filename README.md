# ASCII Video/Image Viewer

Терминальный просмотрщик изображений и видео в ASCII-арте с поддержкой цветов и аудио.

## Возможности

- Поддержка форматов: JPEG, PNG, MP4, AVI, MKV, MOV, WebM
- Цветной ASCII-арт с 256-цветной палитрой
- Плавное масштабирование и навигация
- Поддержка аудио для видеофайлов
- Инкрементальное обновление экрана
- Синхронизация аудио и видео
- Эффекты: дизеринг, инвертирование цветов, зум
- Управление воспроизведением видео
